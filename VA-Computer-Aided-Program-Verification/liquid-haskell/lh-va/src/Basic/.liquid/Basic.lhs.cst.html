Defining Types {#definetype} -------------- Let us define some refinement types:^[You can read the type of `Zero` as: "`v` is an `Int` *such that* `v` equals `0`" and `NonZero` as : "`v` is an `Int` *such that* `v` does not equal `0`"] \begin{code} 
38: {-@ type Zero    = {v:Int | v == 0} @-}
39: {-@ type NonZero = {v:Int | v /= 0} @-}
\end{code} \newthought{The Value Variable} `v` denotes the set of valid inhabitants of each refinement type.
